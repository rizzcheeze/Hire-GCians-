Frontend presentation build for Hire GCians!.

Open `frontend/index.html` to start the standalone demo flow.

Included page groups:
- Public: landing, jobs, employers, about, auth, all-pages index
- Student: dashboard, jobs, applications, saved, profile, resume, settings, public profile
- Employer: dashboard, posting, applicants, listings, hired, company profile, settings
- Admin: overview, users, listings, applications, AI logs, employers, announcements, reports, settings, audit logs

Shared assets live in `frontend/assets/`.
